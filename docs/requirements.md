 Платформа предназначена для фотографов, которые могут загружать и продавать свои фото и видео, и для серферов, которые могут искать, просматривать и покупать этот контент.
 Сайт содержит каталог серфспотов и каждый медиафайл прикреплен к определенному споту.

 Центральный функционал сайта - это удобная карта с поиском по дате и возможностью покупки или добавления контента в избранное или в корзину.

 Язык интерфейса - английский.

 ## 2. Основные участники (Роли)

- **Фотограф (Продавец):** Пользователь, который загружает медиаконтент для продажи.
- **Серфер (Покупатель):** Пользователь, который ищет и покупает медиаконтент.
Пользователь может иметь обе роли одновременно.

  ## 3. Функциональные требования

  ### 3.1. Аутентификация и управление пользователями
  - Пользователи могут регистрироваться и входить с помощью email и пароля.
  - Предусмотрена аутентификация через Google и
  Facebook.
  - Пользователи могут выходить из системы.
  - Покупку / продажу / управление загрузками может выполнять только авторизованный пользователь

  ### 3.2. Управление медиаконтентом (для Фотографов)
  - Фотографы могут загружать медиафайлы (фото и видео) на платформу.
  - **Наложение водяных знаков (Watermarking):** Все превью-версии видео и
  фото должны автоматически защищаться водяным знаком (логотипом
  "Surf-Spy").
  - Фотографы могут устанавливать цену для каждого медиафайла.
  - Фотографы могут публиковать или скрывать свои медиафайлы из общего
  доступа.
  -  Фотографы могут просматривать список своих загруженных медиафайлов.
  -  Фотографы могут удалять свои медиафайлы.

  ### 3.3. Покупка/продажа контента.
  - Пользователи могут оплачивать товары в
  корзине с помощью банковской карты.
  -  После успешной оплаты пользователь получает временную ссылку на оригинальные файлы (без водяного знака) в личном кабинете.
  - Платформа получает процент с продажи.
  - Продавец получает выставленную им сумму за вычетом процента платформы.

  ### 3.4. Поиск
  - Есть каталог серфспотов
  - у каждого серфспота галерея с сортировкой по дате
  - нужна карта со спотами для интерактивности и вовлечения в просмотр
  ### 3.5. Ограничения загрузки файлов (MVP - Free Tier)

  **Cloudinary Free Tier Constraints:**
  - Image upload limit: 10MB per file
  - Video upload limit: 100MB per file
  - Total storage: 25GB
  - Bandwidth: 25GB/month

  **Proposed MVP Limits (Balanced for user comfort):**
  - **Images**: 10MB max (Cloudinary limit)
    - Covers: High-quality JPEGs (4000x3000 @ 90% quality ≈ 5-8MB)
    - Professional RAW files will need compression before upload
  - **Videos**: 50MB max (conservative - allows more uploads before hitting bandwidth)
    - Covers: 1080p 30-second clips (H.264 @ 10Mbps ≈ 40MB)
    - 4K clips will need compression
  - **Batch upload**:
    - Max 20 files per batch (UI performance + preview generation)
    - Max 200MB total per batch (prevents memory issues)
  - **Daily limit per photographer**: 100 uploads/day (prevents abuse, ample for pro shooters)

  **User Experience Compromises:**
  - Show file size warning before upload (not blocking, just informative)
  - Provide compression tips: "Compress RAW to JPEG" or "Use 1080p for videos"
  - Clear error messages when limits exceeded
